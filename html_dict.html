Dicionário de Tags HTML

Tags Utilizadas

  • <!DOCTYPE html>: Define o tipo de documento como HTML5.
  • <html>: Envolve todo o conteúdo da página, informando o início e fim do documento HTML.
  • <head>: Contém metadados da página, como charset e links para estilos.
  • <meta>: Especifica metadados como codificação de caracteres e viewport para a página.
  • <title>: Define o título da página que aparece na aba do navegador.
  • <link>: Conecta arquivos externos, como estilos CSS ou ícones.
  • <body>: Contém todo o conteúdo visível da página, como texto, imagens e navegação.
  • <header>: Define o cabeçalho da página, incluindo navegação e títulos.
  • <nav>: Define uma barra de navegação com links.
  • <a>: Cria um link clicável para navegar entre páginas ou sites.
  • <h1>, <h2>: Títulos de diferentes níveis de importância.
  • <section>: Define seções distintas dentro de uma página.
  • <ul>: Cria uma lista não ordenada.
  • <li>: Define itens dentro de uma lista.
  • <img>: Exibe uma imagem na página.
  • <footer>: Define o rodapé da página com informações de contato e links.
  • <address>: Fornece detalhes de contato ou informações de autor.
